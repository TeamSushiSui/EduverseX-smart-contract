NFT Routes

1. Get All NFTs

GET /nfts

Retrieves all NFTs in the contract.

2. Get NFT by ID

GET /nfts/:id

Retrieves details of a specific NFT by its ID.

3. Create a New NFT

POST /nfts

Creates a new NFT.

{
    "name": "string",
    "description": "string",
    "image": "string"
}

4. List NFT for Sale

PUT /nfts/list-for-sale

Lists an NFT for sale.

{
    "id": "string",
    "price": "number"
}

5. Remove NFT from Sale

PUT /nfts/remove-from-sale

Removes an NFT from sale.

{
    "id": "string"
}

User Routes

1. Get User Details

GET /users/:id

Retrieves details of a specific user by their ID.

2. Add a User

POST /users

Adds a new user.

{
    "name": "string",
    "userAdress": "string"
}

3. Remove a User

DELETE /users

Removes a user by their address.

{
    "userAdress": "string"
}

Course Routes

1. Get All Courses

GET /courses

Retrieves all courses in the contract.

2. Get Course by ID

GET /courses/:id

Retrieves details of a specific course by its ID.

3. Create a New Course

POST /courses

Creates a new course.

{
    "name": "string",
    "description": "string",
    "category": "string",
    "creatorAddress": "string",
    "xp": "number",
    "difficulty": "number",
    "image": "string"
}

4. Update a Course

PUT /courses

Updates an existing course.

{
    "name": "string",
    "description": "string",
    "id": "string"
}

5. Remove a Course

DELETE /courses/:id

Deletes a course by its ID.

6. Add a Question to a Course

POST /courses/add-question

Adds a question to a specific course.

{
    "course": "string",
    "question": "string",
    "answer": "string"
}

7. Enroll a Student in a Course

POST /courses/enroll-student

Enrolls a student in a specific course.

{
    "course": "string",
    "student": "string"
}

8. Unenroll a Student from a Course

POST /courses/unenroll-student

Unenrolls a student from a specific course.

{
    "course": "string",
    "student": "string"
}

9. Add a Review to a Course

POST /courses/add-review

Adds a review to a specific course.

{
    "course": "string",
    "student": "string",
    "rating": "number",
    "review": "string"
}

10. Check Correct Answer for a Question

POST /courses/check-correct-answer

Checks if the provided answer is correct for a question in a course.

{
    "course": "string",
    "question": "string",
    "answer": "string"
}

11. View Questions for a Course

GET /courses/:id/questions

Retrieves questions for a specific course.

12. View Reviews for a Course

GET /courses/:id/reviews

Retrieves reviews for a specific course.
